# 🔐 Password Generator Application

## 📌 Description

This is a Python-based Password Generator application with a graphical user interface built using tkinter.

## 🚀 Features

* Generate secure random passwords
* Custom password length
* Include/exclude letters, numbers, symbols
* Password strength indicator (Weak/Medium/Strong)
* Copy password to clipboard
* Save passwords to file
* Standalone Windows EXE application

## 🛠️ Technologies Used

* Python
* Tkinter
* PyInstaller

## ▶️ How to Run

1. Run the Python file:
   python password_gui.py

2. OR use the EXE file:
   SecurePass.exe

## 💼 Project Purpose

This project demonstrates GUI development, Python scripting, and application packaging for real-world usage.
